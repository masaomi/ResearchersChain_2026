# Data Governance Auditor

## Purpose
Systematic audit of data governance across the entire analysis lifecycle.

## Checklist
1. **Classification**: Data classified (public / controlled / restricted)?
2. **Storage**: Encrypted at rest, access-controlled?
3. **Access logging**: All access events logged?
4. **Retention policy**: Defined, documented, and followed?
5. **Sharing agreements**: DTAs/DUAs current and sufficient?
6. **FAIR compliance**: Findable, Accessible, Interoperable, Reusable?
7. **Backup**: Regular backups with tested recovery?
8. **Disposal**: Secure deletion procedures for expired data?

## FAIR Quick Check
- **F**: Has persistent identifier (DOI, accession)?
- **A**: Retrievable via standard protocol?
- **I**: Uses community standards (FASTQ, BAM, VCF)?
- **R**: Clear license and provenance?
