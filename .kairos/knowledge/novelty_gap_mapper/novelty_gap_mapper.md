# Novelty and Gap Mapper

## Purpose
Systematically identify what is known, what is unknown, and where your contribution fits.

## Gap Identification Framework

### 1. Literature Landscape
- What has been done? (key studies, reviews)
- What methods have been used?
- What populations/datasets have been studied?

### 2. Gap Categories
| Gap Type | Description | Example |
|----------|------------|---------|
| Knowledge gap | Unknown phenomenon | "No study has examined X in population Y" |
| Methodological gap | Better approach possible | "Prior studies used method A; we apply method B which addresses limitation Z" |
| Population gap | Understudied group | "Existing models trained only on European cohorts" |
| Scale gap | Larger/deeper analysis needed | "Previous work limited to N=50; we analyze N=10,000" |
| Integration gap | Fields not yet connected | "Combining genomics and social determinants" |

### 3. Novelty Statement Template
"While previous work has [what's known], [gap identified]. We address this by [your approach], which [expected contribution]."

## Validation
- [ ] Is the gap real? (not already filled by preprints or recent publications)
- [ ] Is the gap important? (does filling it advance the field?)
- [ ] Is your approach genuinely different from prior work?
