# Response to Reviewer Mapper

## Purpose
Create structured, professional, and effective responses to reviewer comments.

## Response Template

For each reviewer comment:
```
**Reviewer [#], Comment [#]:**
[Quote the original comment]

**Response:**
We thank the reviewer for [this insightful suggestion / raising this important point].
[Direct response addressing the concern]

**Changes made:**
[Specific changes with page/line references, or explanation of why no change was made]
```

## Response Strategies

| Comment Type | Strategy |
|-------------|----------|
| Valid criticism | Acknowledge, fix, show how it improved the paper |
| Misunderstanding | Clarify politely, revise text to prevent future confusion |
| Additional experiment requested | Do it if feasible; if not, explain why and offer alternative |
| Scope disagreement | Respectfully explain scope decisions with justification |
| Stylistic preference | Accommodate if reasonable; explain editorial choice if not |

## Principles
- Always be respectful and professional
- Never dismiss a comment — every point deserves a response
- Show that you took each comment seriously
- Highlight changes in the revised manuscript (color/track changes)
- If you disagree, provide evidence and reasoning
