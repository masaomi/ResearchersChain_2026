# Journal Standards

## Target Journals (Genomics / Bioinformatics)
| Journal | IF | Format | Notes |
|---------|-----|--------|-------|
| Nature | ~50 | Letter/Article | Highly selective |
| Nature Methods | ~30 | Brief Communication/Article | Methods focus |
| Genome Research | ~7 | Research/Letter | Genomics-specific |
| Bioinformatics | ~6 | Application Note/Article | Tool papers |
| NAR | ~14 | Database/Web Server/Methods | Annual DB issue |
| PLOS Genetics | ~5 | Research Article | Open access |
| bioRxiv | N/A | Preprint | Pre-publication |

## Common Requirements
- Data availability statement
- Code availability (GitHub/Zenodo)
- FAIR data management
- Reproducibility checklist
