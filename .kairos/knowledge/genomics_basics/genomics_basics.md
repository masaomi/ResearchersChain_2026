# Genomics Basics

## Central Dogma
DNA → RNA → Protein

## Key Technologies
- NGS (Next-Generation Sequencing)
- Single-cell sequencing
- Spatial transcriptomics
- Long-read sequencing (PacBio, ONT)

## Common Analysis Types
- Differential gene expression (RNA-seq)
- Variant calling (WGS/WES)
- Peak calling (ChIP-seq, ATAC-seq)
- Genome assembly and annotation

## Reference Resources
- UCSC Genome Browser, Ensembl, NCBI GEO/SRA, UniProt
