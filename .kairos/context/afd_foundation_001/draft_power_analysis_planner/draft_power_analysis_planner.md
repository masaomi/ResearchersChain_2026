# Power Analysis Planner (Draft)
## Workflow
1. Define primary endpoint and expected effect size
2. Set significance level (alpha) and desired power (1-beta)
3. Consider study design (one/two-sided, paired/unpaired, multiple groups)
4. Calculate minimum sample size using appropriate method (G*Power, pwr package)
5. Assess feasibility and adjust if needed (budget, available samples)
6. Output: sample size recommendation with power curve and assumptions
