# Data Governance Auditor (Draft)
## Checklist
1. Data classification (public / controlled / restricted)
2. Storage compliance (encrypted at rest, access-controlled)
3. Access logging enabled
4. Data retention policy defined and followed
5. Data sharing agreements current and sufficient
6. FAIR compliance check (Findable, Accessible, Interoperable, Reusable)
7. Output: governance compliance report with gap analysis
